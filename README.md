# About

Lead platform engineer with 17+ years building scalable, high-performance infrastructure on AWS — the kind that goes from zero to millions of users and stays cheap and boring to run.

The part I enjoy most is what comes after the work is done: turning hard-won production experience into clean, reusable references other people can build on, and shipping small, well-made tools. Reusable AWS Organizations guardrails, an OpenTofu + EKS reference architecture, a self-hosted full-text RSS tool, a calendar utility on the App Store. Build it properly, document the sharp edges, give it away to the community.

You can find me as **@mijndert** most places, with a longer home at [mijndertstuij.nl](https://mijndertstuij.nl/).

# Selected work

- **[aws-scp-catalog](https://github.com/mijndert/aws-scp-catalog)** — Reusable OpenTofu for AWS Organizations Service Control Policies. Common guardrails and a flexible attachment structure.
- **[aws-tf-reference](https://github.com/mijndert/aws-tf-reference)** — A reference architecture for AWS: VPC and an EKS cluster on OpenTofu, with an example deployment.
- **[rss-fulltext](https://github.com/mijndert/rss-fulltext)** — Rewrites headline-only feeds into full-text RSS, Atom, and JSON. Self-hosted, served as static files, container-hardened by default.
- **[CalAction](https://apps.apple.com/us/app/calaction/id6760361082)** — Calendar actions for iOS via App Intents. Available on the App Store.
